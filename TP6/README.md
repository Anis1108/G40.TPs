# TP6

Dans ce TP on a vu comment fonctionne un serveur web en Python puis on a utilisé Django.

Au début on a fait un serveur HTTP simple :
on a appris à gérer les requêtes, afficher une page HTML (index.html) et gérer une erreur 404.

Ensuite on est passé à Django :
- création du projet chatServer
- création de l’app g40aChat
- ajout dans settings.py
- création d’une vue qui affiche un message
- configuration des urls

On a lancé le serveur et testé sur :
http://127.0.0.1:8000/chat/

À la fin on a fait deux petits exercices en Python :
- escaliers (logique récursive)
- compter le nombre de mots avec split()

Ce TP m’a permis de mieux comprendre comment fonctionne un serveur web et les bases de Django.
